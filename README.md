# LiveChat
Тестовое задание: cоздать «живой» чат с использованием встроенных методов Rails 7

## Детали
- Ruby "3.1.3"
- Rails "7.0.5.1"
- SQLite3
- Importmap
- Stimulus
- Tailwindcss
- Minitest
- Devise

## Установка
Установить зависимости и базу данных:
```
$ bin/setup
```

## Запуск
Запустить сервер приложения:
```
$ bin/dev
```

Перейти по адресу в адресной строке браузера:
```
localhost:3000
```

Чатиться :relaxed:
